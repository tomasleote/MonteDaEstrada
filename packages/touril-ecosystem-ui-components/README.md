# @touril-ecosystem/ui-components

Shared React component library used across all Touril Ecosystem websites.

## Planned Contents

- `src/components/` — shared components (Header, Footer, etc.)
- `src/styles/` — shared SCSS design tokens (_variables.scss, _mixins.scss)

## Usage (once components are migrated)

In any app within this monorepo:

```js
import { Header } from '@touril-ecosystem/ui-components';
```

## Current Status

Scaffold only — component migration from `apps/monte-da-estrada` is a future task.
